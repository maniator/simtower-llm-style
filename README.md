# SimTower LLM Style

A collection of tower simulation game implementations built with different AI-assisted development approaches.

## 📁 Project Structure

This project contains three distinct implementations of a SimTower-style building simulation game:

- **copilot/** - Tower simulation built with GitHub Copilot
- **gemini/** - Tower simulation built with Google Gemini AI
- **github-copilot/** - Tower simulation built with GitHub Copilot (alternative implementation)

## 🎮 Game Overview

All implementations feature a tower building simulation game inspired by the original SimTower (1994) with the following core mechanics:

### Features
- **Building Management**: Construct apartments, offices, shops, restaurants, hotels, lobbies, and elevator shafts
- **Economics**: Manage finances through income from various room types
- **Population**: Monitor and grow your tower's resident population
- **Rating System**: Achieve higher star ratings through effective tower management
- **Time Controls**: Play, pause, and fast-forward simulation speed

### Game Elements

**Room Types:**
- Apartment - Residential housing (500 cost, 50 income/day)
- Office - Commercial space (800 cost, 90 income/day)
- Shop - Retail space (700 cost, 80 income/day)
- Restaurant - Food service (1000 cost, 120 income/day)
- Hotel Room - Temporary housing (900 cost, 110 income/day)
- Lobby - Entrance area (300 cost)
- Elevator Shaft - Vertical transportation (400 cost)

## 🚀 Getting Started

1. Open `index.html` in your browser to see the landing page
2. Click on any implementation (copilot, gemini, or github-copilot) to play
3. Use the sidebar to select building types
4. Click on the tower to place rooms
5. Monitor your stats: Money, Population, Rating, and Time

## 📋 File Structure

Each implementation contains:
- `index.html` - Game UI and interface
- `main.js` - Core game logic and simulation engine
- `style.css` - Visual styling and layout

## 💡 AI-Assisted Development

This project demonstrates tower simulation development using different LLM assistants:

- **GitHub Copilot** implementations showcase collaborative AI coding with a familiar IDE assistant
- **Google Gemini** implementation explores building games with alternative generative AI
- Each implementation may have different code structures, optimizations, or feature sets based on its development approach

## 🎯 Objectives

When building/playing:
- Maximize income while maintaining population happiness
- Increase your tower's star rating
- Balance construction costs with revenue potential
- Manage elevator access to keep residents accessible
- Create a thriving, self-sustaining tower community
